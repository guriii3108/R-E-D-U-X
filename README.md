# Pixly

Pixly is a powerful media discovery application that lets you search and explore a vast collection of images, videos, and GIFs. With a sleek and intuitive interface, Pixly makes it easy to find visual content for any purpose. Experience seamless searching and high-quality media at your fingertips.

## Features

-   **Search**: Instantly find images, videos, and GIFs.
-   **Explore**: Discover trending and popular media.
-   **High Quality**: Access high-resolution assets.
-   **Fast & Responsive**: Built with React and Redux for optimal performance.

## Getting Started

1. Clone the repository.
2. Run \`npm install\` to install dependencies.
3. Run \`npm run dev\` to start the development server.
